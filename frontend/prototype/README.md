# Prototype

Este prototipo es una maqueta estatica inicial de Kobra. No usa framework ni dependencias; sirve para validar direccion visual, jerarquia de informacion y tono de producto antes de iniciar Next.js.

## Archivos

- `index.html`: landing publica para explicar producto, problema, flujo, MVP y propuesta comercial.
- `dashboard.html`: demo del dashboard operativo para visualizar la experiencia del producto.
- `styles.css`: tokens visuales, layout responsive y componentes base.

## Como verlo

Abre `index.html` en el navegador o desde VSCode con Live Preview. Para revisar la demo interna, abre `dashboard.html`.

## Criterios aplicados

- Landing clara con blanco como base y narrativa de producto real.
- Dashboard financiero/data-dense con monitoreo en tiempo real.
- Paleta sobria basada en blanco, superficies calidas, ambar margen, cobre riesgo y azul acero.
- Sin emojis, sin stock photos, sin hero generico SaaS, sin lenguaje de IA inflado.
- Recomendaciones con evidencia visible y decision humana.
